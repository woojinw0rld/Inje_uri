# A 개발자 남은 작업

구현 완료된 A API 명세와 서버 레이어 구조 설명은 프로젝트에 반영되어 이 문서에서 제거했다.

## 남은 정책 확정 항목

- QR_LIMITED 접근 레벨의 실제 제한 범위 확정
- 탈퇴 정책 세부 확정
  - 재가입 가능 여부
  - 재가입 대기기간 존재 여부
  - WITHDRAWN 처리 후 개인정보 보존 범위
- 닉네임 변경 주기 정책 확정
- 성별 및 추천 대상 모델 확정
- 온보딩 완료 조건 확정
  - 필수 입력 필드
  - 프로필 이미지 최소/최대 개수
  - 대표 이미지 필수 여부
- 공통 노출 정책 확정
  - 차단 후 추천/프로필/스토리/채팅 비노출 범위
  - 신고 접수 후 자동 블라인드 기준
- 시간 정책 확정
  - DB 저장 시간 기준
  - 만료 계산 기준
  - batch 처리와 write-path 처리 기준
