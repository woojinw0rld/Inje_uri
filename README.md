# 인제우리 Next.js 모바일 프로토타입

모바일 중심으로 다시 정리한 App Router 프로젝트입니다.

## 실행

```bash
npm install
npm run dev
```

## 구조

- `src/app`: 라우트
- `src/components`: 화면/공용 컴포넌트
- `src/data`: 목업 데이터 및 키워드 카탈로그
- `src/lib/repositories`: 백엔드 연동 전 교체 가능한 저장소 계층
- `src/app/api`: 백엔드 시작용 기본 API 라우트

## 핵심 반영사항

- 데스크톱에서는 **가운데 정렬된 앱 컬럼**
- 모바일에서는 **기기 폭 전체 활용**
- 커뮤니티 제거, **셀프소개팅 24시간 스토리형** 반영
- `나를 좋아하는 사람` 화면에서 **바로 채팅 생성** 가능
- 활성 채팅방 **최대 5개 제한**
- 프로필 편집은 **직접 입력이 아닌 선택형 키워드 편집**
