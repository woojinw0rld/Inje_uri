인제우리 (Inje_uri)
인제대학교 재학생 전용 소개팅 모바일 웹 앱. Next.js App Router 기반, 모바일 퍼스트 프로토타입.

브랜치 전략
  main
  └── dev
      ├── feature/기능이름
      ├── feature/기능이름
      └── ...
브랜치	용도
main	배포용. 검증 완료된 코드만 병합
dev	통합 개발 브랜치. feature 병합 후 검증
feature/기능이름	기능 단위 작업 브랜치. dev에서 분기
흐름

dev → feature/기능이름 분기
기능 개발 완료 후 dev에 PR & merge
dev에서 통합 검증 완료 후 main에 PR & merge
main 직접 push 금지. 반드시 dev를 거쳐야 함.
